# 在 Windows 7 上安装 YcyzClass

> [!warning]
> **不建议在 Windows 10 以下的操作系统上运行 YcyzClass。** 

> [!caution]
> 此页面上的方法（设置环境变量）存在兼容性问题（[#309](https://github.com/YcyzClass/YcyzClass/issues/309)），如果您有在 Windows 7 上安装 YcyzClass 的需求，请使用 [.NET 6 兼容版本](https://github.com/YcyzClass/YcyzClass-net6/)。如果您先前使用本页面的方法通过设置环境变量来兼容 YcyzClass，请回滚更改。
